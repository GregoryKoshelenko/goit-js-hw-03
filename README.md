# GoIT JavaScript — Домашня робота 3

Три завдання з масивів і рядків. Результати перевірки виводяться в консоль браузера після відкриття `index.html`.

## Структура проєкту

- `index.html` — сторінка з підключеними скриптами
- `js/task-1.js` — генератор slug (`slugify`)
- `js/task-2.js` — композиція масивів (`makeArray`)
- `js/task-3.js` — фільтрація чисел (`filterArray`)

## Локальний запуск

Відкрийте `index.html` у браузері або запустіть локальний сервер у корені репозиторію.

## Форматування коду

```bash
npm install
npm run format
```

## Здача на перевірку

Після публікації репозиторію на GitHub вкажіть у здачі:

1. **Посилання на репозиторій** з вихідним кодом
2. **Жива сторінка на GitHub Pages** (налаштування: Settings → Pages → Branch, зазвичай `main` / `/ (root)`)

Додатково прикріпіть **zip-архів** репозиторію згідно з інструкцією курсу.
